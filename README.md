## 🧪 Conceptos Prácticos de ASIR
*Resúmenes rápidos para repasar antes del examen.*

### 📂 Modelo OSI vs TCP/IP
| Capa | Función | Dispositivo |
| :--- | :--- | :--- |
| **Física** | Bits y cables | Hub / Repetidor |
| **Enlace** | Direcciones MAC | Switch |
| **Red** | Direcciones IP | Router |

### 🔢 Subnetting (Sin jaleos)
* **Máscara de red:** El "filtro" que decide quién es de mi familia.
* **Gateway (Puerta de enlace):** La salida al mundo exterior.
